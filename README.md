# Woods API

1. Levantar la base de datos e instalar dependencias:

```
docker-compose up -d
```

npm i @nestjs/config
npm i --save @nestjs/typeorm typeorm pg

docker exec -it woodsdb psql -U postgres

npm install class-validator class-transformer
npm i uuid
npm i --save-dev @types/uuid

npm run start:dev
